# PracticeFit

This project was generated with Angular version 14.2.10 and Node version 18.15.0. you can run application on your local machine using following steps given below.

### Installation Step 1

Install [NodeJs](https://nodejs.org/en/blog/release/v18.15.0) version 18.15.0.

| Package Name | Installation Source                                                   |
| ------------ | --------------------------------------------------------------------- |
| NodeJs       | [NodeJs version 18.15.0](https://nodejs.org/en/blog/release/v18.15.0) |

### Installation Step 2

Now installing [Angular CLI](https://www.npmjs.com/package/@angular/cli/v/14.2.10) version 14.2.10, run `cmd` as administrator then type below commands.

```sh
npm install @angular/cli@14.2.10 -g
npm install @angular/cli@14.2.10
```

### Installation Step 3

Open project source code in VS Code and open VS Code terminal and hit command `npm install` this will help to load all dependencies packages to run the application.

```sh
npm install
```

### Installation Step 4

After installing all dependencies of project open VS Code terminal and hit command `ng serve -o`

```sh
ng serve -o
```

### Finalize View Application in Browser

After successful running application browser will serve the application on `http://localhost:4200/` you can view application in your browser.

```sh
http://localhost:4200/
```

## Build for Production

Run `ng build` to build the project. The build artifacts will be stored in the `dist/` directory.

```sh
ng build
```

## Running unit tests (optional)

Run `ng test` to execute the unit tests via [Karma](https://karma-runner.github.io).

```sh
ng test
```

## Running end-to-end tests (optional)

Run `ng e2e` to execute the end-to-end tests via a platform of your choice. To use this command, you need to first add a package that implements end-to-end testing capabilities.
